# Initial Concept
ApartmentHunter: A high-performance apartment search decision matrix and tracker built with React, Vite, and Tailwind CSS v4. Modularized for scalability and fully integrated with Gemini AI for smart listing extraction and scam detection.

# Product Guide: ApartmentHunter

## Vision
To provide a high-performance, intelligent apartment search decision matrix that simplifies the apartment-hunting process through automation, smart data extraction, and weighted comparisons.

## Target Audience
- Individuals or families actively searching for a new rental property.
- Users who want to compare multiple listings side-by-side using custom criteria.
- Tech-savvy users looking for AI-enhanced tools to filter out scams and extract listing data efficiently.

## Core Features
- **AI Smart Import**: Seamless extraction of structured data (price, beds, sqft, location, etc.) from unstructured text listing descriptions.
- **Scam Detection**: Intelligent flagging of suspicious rental listings using market benchmarks and fraud patterns.
- **Commute Matrix**: Real-time travel time estimations to key destinations (work, school, gym, etc.) integrated into the listing overview.
- **Weighted Ranking System**: A dynamic decision matrix that ranks listings based on user-defined priorities (e.g., price vs. commute vs. size).
- **Listing Management**: Track status, notes, and contact information for multiple apartment candidates.
- **Data Export**: Export listing data to CSV for offline analysis or archival.

## Design Goals
- **High Performance**: A responsive and fast UI built with Vite and React.
- **Clarity & Utility**: Focus on data density and legibility for decision-making.
- **Accessibility**: Adhere to modern web accessibility standards.
- **Scalability**: A modular architecture that allows for easy addition of new features and AI capabilities.
